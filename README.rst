WhiFuN
======

WhiFuN is a GUI-based MATLAB toolbox for white matter (WM) and gray matter (GM)
functional connectivity analysis in fMRI.

It provides tools for:

- Automated preprocessing of WM and GM fMRI data (SPM12-based)
- Construction of WM and GM Functional Networks (FNs) using K-means clustering
- Functional connectivity (FC) analysis across WM, GM, corpus callosum, and atlases
- Statistical analysis and visualization workflows for group-level studies
